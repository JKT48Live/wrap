JKT48 Wrapped Extension

Install

Kompatibel Browser

Ekstensi ini bisa diinstal pada browser berbasis Chromium, misalnya:

  • Google Chrome
  • Microsoft Edge
  • Brave, dan lainnya.

Android & iOS

Android & iOS bisa menggunakan browser ini

Cara Menggunakan

1. Buka dan login pada website JKT48.
2. Klik ekstensi JKT48 Wrapped.
3. Pilih tahun.
4. Klik "Get Data".
